Actualize property ExtInfo (Extension Info).

public Ext_PrepareInfo([mixed $Ext = false ]) : mixed

ExtInfo will be an array with keys 'load', 'br', 'ctype' and 'pic_path'. Keys 'rpl_what' and 'rpl_with' are optional. load: files in the archive to be automatically loaded by OpenTBS when the archive is loaded. Separate files with comma ';'. br: string that replace break-lines in the values merged by TBS, set to false if no conversion. frm: format of the file ('odf' or 'openxml'), for now it is used only to activate a special feature for openxml files ctype: (optional) the Content-Type header name that should be use for HTTP download. Omit or set to '' if not specified. pic_path: (optional) the folder nale in the archive where to place pictures rpl_what: (optional) string to replace automatically in the files when they are loaded. Can be a string or an array. rpl_with: (optional) to be used with 'rpl_what', Can be a string or an array. User can define his own Extension Information, they are taken in acount if saved int the global variable $_OPENTBS_AutoExt.
